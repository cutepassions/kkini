# 서비스 소개 🚩
* 자취 요리 가이드 및 공유 SNS 서비스입니다.
* 일상의 가장 가까운 부분인 식사를 공유하며 공감대를 형성하고 싶어 개발을 시작했습니다.
* 치솟는 물가에 허덕이는 주머니 사정을 지킬 수 있습니다!
* 당신의 요리 실력을 뽐내보세요!
